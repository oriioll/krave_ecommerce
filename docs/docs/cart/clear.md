# Clear Entire Cart

<span class="api-badge api-badge--delete">DELETE</span> `/cart`

Removes all products from the user's active shopping cart, leaving it empty.

### Responses

#### `200 OK`

```json
{
  "status": "success",
  "message": "Product cart emptied successfully"
}
```
